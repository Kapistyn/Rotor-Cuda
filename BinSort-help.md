## For Linux
- Open BinSort Folder
  
  ```sh

    $ cd BinSort 

    $ sudo apt install -y libgmp-dev
    
    $ make

    ```
- Add ' hash160_out.bin ' ( generated by addresses_to_hash160.py ) for sorting*
  
  ```sh
 
    $ ./BinSort 20 hash160_out.bin h160_sorted.bin
    
    ```
- Now move the ' h160_sorted.bin ' to Linux folder
